# ChatUp overview

ChatUp je jednoduchá aplikace pro oboustrannou komunikaci za využití WebSocket package v Node.js

## Požadavky ke spuštění

Pro spuštění je potřeba:
1. Jakýkoliv prohlížeč kompatibilní s WebSocket API (viz. [dokumentace](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API#browser_compatibility)).
2. Node.js v22.0.0+
3. NPM

## Spuštění
Po stažení repositáře nainstalujte potřebné npm balíčky pomocí npm i a otevřete alespoň 2 instance index.html v prohlížeči.
Samostatně spustit server.js přes node.
